# HTML/CSS研修 実習用repo with Nuxt.js

## Get Started

1. [Git](https://git-scm.com)および[Node.js](https://nodejs.org)をインストール
2. GitHubにログインし、このテンプレートリポジトリをもとにリポジトリ作成
3. `npm install`
4. `npm run dev`

## 概要

本リポジトリは、[Nuxt.js](https://nuxt.js)をベースに構築した研修用リポジトリである。  
HTMLとCSSの基礎基本～実践まで体系的に学べるように構成した。  
また、今回はHTMLとCSSを同じファイルにかけ、<head>タグを省略できるため、vue.jsベースのNuxt.jsを採用した。
